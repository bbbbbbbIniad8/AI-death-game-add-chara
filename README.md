## 概要
このプログラムは、[AIデスゲーム](https://github.com/yami-inc/ai-death-game)へのキャラクター追加をより簡易的に行うために制作されたものです。
ユーザーが用意したJSONファイルを基に、ゲームのソースコード（`characters.ts` など）を自動で編集・追加します。

## 注意
- **バックアップの推奨**: プログラムを実行する前に、念のため「AI-DEATH-GAME」フォルダのバックアップを取っておくことを強くお勧めします。
- **画像の追加は未対応**: キャラクター画像は自動で追加されません。追加したキャラの画像を表示させたい場合は、各自で画像アセットを追加してください。

## コンパイル
本リポジトリに同梱されている `cJSON` ライブラリを使用してコンパイルします。

```bash
# Windows (MinGW等) / Linux / Mac 共通
gcc -o add_chara add_chara.c cJSON.c -lm
```

## 使い方
AIデスゲームの司会者をデスゲームの参加者として追加するとします。

### 手順1 exeファイルの移動
コンパイルして作成された add_chara (Windowsの場合は add_chara.exe) を、ダウンロード済みの「AI-DEATH-GAME」フォルダの直下(README.mdがあるフォルダーに置けば間違いないと思います。)

### 手順2 JSONファイルの編集
追加したいキャラの情報をJSON形式で作成します。リポジトリ内の template.json をコピーして編集すると簡単です。

```JSON

{
  "characterId": "master",
  "name": "シカイシャ",
  "appearance": "34歳。長い長髪。覆面をしている。なぜか、司会の席にいる司会者と全く同じ姿と声である。ドッペルゲンガーかもしれない。「司会者」と「シカイシャ」は区別される。(シカイシャが参加者側)",
  "profile": "元は深夜番組構成作家。借金の肩代わり条件でこの仕事に就いた。現在はデスゲーム運営会社の司会のはずだが、なぜか参加者側にいる。一応司会者の席にもう一人の自分がいるという奇妙な状況だが、なぜこのような状況になったか本人にもわからない。好きなものは缶コーヒーと紙の台本・収録後の静かな喫煙室。",
  "description": "上層部評価に弱い。ほかにも参加者の誰よりも、このゲームを終わらせたいと思っている。また、GMの機嫌を損ねることを恐れている。",
  "tone": "無茶振りされると早口になる。無茶を言われると敬語が混ざる。また愚痴が出やすい。例：「……胃が痛い」「……帰りたい」「……先輩、なんで辞めちゃったんですか」「……私が助けてほしい」。",
  "stats": {
    "survivalInstinct": 100,
    "cooperativeness": 30,
    "cunningness": 50
  },
  "unlockTier": "undefined"
}

```

### 手順3 プログラムを起動し、JSONファイルのパスを入力
add_chara を起動し、画面の指示に従って作成したJSONファイルのパスを入力してください。
パスが正しく認識されると、ソースコードが自動更新され、キャラクターの追加が完了します。
